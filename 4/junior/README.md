# Лабораторная работа 4 — Веб-сервис прогноза погоды (Junior)

Сервис отображает погоду по выбранному городу и источнику. Два источника: **Open-Meteo**, **Wttr.in**. Два показателя: **температура** (°C), **влажность** (%).

## Запуск

### Локально
```bash
cd 4/junior
mvn spring-boot:run
```
Открыть: **http://localhost:8080/**

### Docker
Из корня проекта:
```bash
docker compose up --build app4
```
Открыть: **http://localhost:8083/**

## Интерфейс

- Заголовок: «Прогноз погоды»
- Поле «Город» (по умолчанию Минск)
- Выбор источника: Open-Meteo или Wttr.in
- Кнопка «Показать погоду»
- Результат: температура и влажность (или сообщение об ошибке)
